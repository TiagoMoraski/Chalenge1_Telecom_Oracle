## 3. Transforma√ß√£o (Transform)
Ap√≥s a extra√ß√£o, identificou-se que quatro colunas (`customer`, `phone`, `internet`, `account`) continham estruturas aninhadas (dicion√°rios armazenados como texto).

**Principais etapas realizadas**
1. Expans√£o das colunas aninhadas usando `pd.json_normalize` e `map(parse_obj)` para lidar com c√©lulas contendo tanto strings quanto dicion√°rios.
2. Adi√ß√£o de prefixos (`customer_`, `phone_`, `internet_`, `account_`) para evitar conflitos de nomes.
3. Convers√£o de tipos:
   - `Churn` transformado para valores bin√°rios (Yes ‚Üí 1, No ‚Üí 0).
   - `account_Charges.Total` convertido para `float`.
4. Limpeza de espa√ßos extras em colunas de texto.
5. Verifica√ß√£o de nulos e padroniza√ß√£o de tipos.

**C√≥digo simplificado**
```python
import pandas as pd
import ast
import json

def parse_obj(x):
    if pd.isna(x):
        return {}
    if isinstance(x, dict):
        return x
    if isinstance(x, str):
        try:
            return ast.literal_eval(x)
        except Exception:
            try:
                return json.loads(x)
            except Exception:
                return {}
    return {}

customer_df = pd.json_normalize(df['customer'].map(parse_obj)).add_prefix('customer_')
phone_df    = pd.json_normalize(df['phone'].map(parse_obj)).add_prefix('phone_')
internet_df = pd.json_normalize(df['internet'].map(parse_obj)).add_prefix('internet_')
account_df  = pd.json_normalize(df['account'].map(parse_obj)).add_prefix('account_')

df_expanded = pd.concat(
    [df[['customerID','Churn']], customer_df, phone_df, internet_df, account_df],
    axis=1
)

## 4. An√°lise Explorat√≥ria (EDA)
Com o dataset limpo e padronizado, foi realizada a An√°lise Explorat√≥ria para identificar padr√µes associados ao churn.

**Principais an√°lises e gr√°ficos gerados**
1. **Distribui√ß√£o geral do churn**  
   - Aproximadamente X% dos clientes sa√≠ram (Churn = 1).
   - Gr√°fico: `countplot` mostrando o equil√≠brio entre clientes retidos e evadidos.

2. **Churn por g√™nero**  
   - Pequena varia√ß√£o entre masculino e feminino, sem impacto significativo.

3. **Churn por tipo de contrato**  
   - Contratos mensais apresentaram taxa de churn significativamente maior.
   - Contratos anuais ou de dois anos ret√™m mais clientes.

4. **Impacto de vari√°veis num√©ricas**  
   - **Tempo de perman√™ncia (tenure)**: clientes novos saem mais.
   - **Mensalidade**: planos mais caros t√™m maior churn.
   - **Total gasto**: clientes com gasto total baixo tamb√©m t√™m alta evas√£o (pouco tempo de perman√™ncia).

5. **Servi√ßos contratados**
   - Fibra √≥ptica tem maior churn que DSL.
